# 2️⃣ CARTES DASHBOARD – EFFET GLASS

Améliorer les cartes statistiques avec un **glass effect moderne** mais **totalement lisible**.

Règles :

* fond semi-transparent
* léger blur
* bordure subtile
* contraste élevé pour texte

Exemple visuel attendu :

* glass propre
* texte très lisible
* icônes visibles

Les cartes doivent rester **professionnelles et institutionnelles**.

---

# 3️⃣ ANIMATION DES CHIFFRES (COUNTING EFFECT)

Tous les indicateurs statistiques doivent avoir un **effet de comptage animé**.

Exemples :

* nombre d’utilisateurs
* nombre de transactions
* volumes
* statistiques dashboard

Animation :

0 → valeur finale
durée : 0.8s à 1.2s

Animation fluide et discrète.

---

# 4️⃣ ICÔNES DYNAMIQUES

Les icônes du dashboard doivent être légèrement animées.

Types d’animations autorisées :

* hover léger
* micro rotation
* pulse discret
* scale léger

Objectif :

rendre l’interface plus vivante sans distraire.

---

# 5️⃣ SURVOL ANIMÉ (HOVER INTERACTIONS)

Tous les éléments interactifs doivent avoir un hover :

Boutons
Cartes
Tableaux
Actions

Effets possibles :

* élévation légère
* ombre dynamique
* changement subtil de couleur
* scale 1.02

Durée animation :

150ms – 250ms

---

# 6️⃣ INTERACTIONS DYNAMIQUES

Ajouter des micro-interactions :

Boutons :

* animation au clic
* feedback visuel

Cartes :

* léger tilt ou lift au hover

Tableaux :

* highlight ligne au survol

Menus :

* transitions fluides

---

# 7️⃣ TECHNOLOGIE RECOMMANDÉE

Utiliser :

Framer Motion

ou animations CSS modernes.

Les animations doivent être :

* fluides
* légères
* performantes

Aucune animation lourde.

---

# 8️⃣ PERFORMANCE

Les animations ne doivent jamais :

* ralentir l’application
* bloquer le scroll
* perturber la lecture

Priorité :

fluidité + lisibilité.

---

# 9️⃣ ACCESSIBILITÉ

Respecter :

* lisibilité du texte
* contraste élevé
* pas de clignotement agressif

Les animations doivent rester **subtiles et professionnelles**.
